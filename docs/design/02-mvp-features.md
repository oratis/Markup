# MVP 功能清单

按"必须 / 应当 / 可以 / 不做"四档划分。MVP = 必须 + 应当。

## P0 — 必须（M1 完成）

### 编辑核心
- [ ] WYSIWYG 编辑（Milkdown）
- [ ] 标准 GFM：标题、加粗、斜体、删除线、链接、图片、列表、引用、分隔线
- [ ] 表格（编辑、新增行列、删除行列）
- [ ] 代码块（语言选择、Shiki 高亮）
- [ ] 任务列表（- [ ] / - [x]）
- [ ] 内联代码
- [ ] 撤销 / 重做
- [ ] 复制粘贴（含跨应用）
- [ ] **Source Mode 切换**（按 ⌘/ 切到 CodeMirror 6 源码视图）

### 文件管理
- [ ] 打开单个文件
- [ ] 打开 vault（文件夹）
- [ ] 文件树（虚拟滚动；支持上千文件）
- [ ] 新建文件 / 新建文件夹
- [ ] 重命名 / 删除（送回收站，不直接删）
- [ ] 拖拽改顺序 / 改父级
- [ ] 最近文件列表
- [ ] **多 tab**（同时打开多个文档）

### 自动保存与同步
- [ ] 编辑 debounce 300ms 自动保存
- [ ] FS 监听：外部修改时提示重新加载
- [ ] 防覆盖：保存前比较 mtime

### macOS 集成
- [ ] 标准菜单栏（File / Edit / View / Format / Window / Help）
- [ ] 完整快捷键（⌘N / ⌘O / ⌘S / ⌘W / ⌘F / ⌘P 等）
- [ ] `.md` 文件关联（双击在 markup 里打开）
- [ ] 暗黑模式跟随系统
- [ ] Dock 图标 + 启动画面

### 性能基线
- [ ] 5MB 文档打开 < 500ms
- [ ] 输入延迟 < 16ms
- [ ] 1 万文件 vault 索引 < 5s
- [ ] 内存常驻 < 300MB

## P1 — 应当（M2 完成）

### Typora 心流功能
- [ ] **Focus Mode**（仅当前段亮，其他段灰化）
- [ ] **Typewriter Mode**（光标始终在屏幕中央）
- [ ] **大纲面板**（侧边栏，标题树，点击跳转）
- [ ] **字数统计**（状态栏，含选中字符数）

### 搜索
- [ ] 当前文件搜索（⌘F）+ 替换（⌘⇧H）
- [ ] **全 vault 搜索**（Tantivy；⌘⇧F）
- [ ] **Quick Open**（⌘P，文件名模糊匹配）
- [ ] **命令面板**（⌘⇧P；执行任意命令）

### Markdown 扩展
- [ ] **数学公式**（KaTeX；行内 $...$ 与块级 $$...$$）
- [ ] **Mermaid 图表**
- [ ] **Front matter**（YAML，识别但不解释）
- [ ] **脚注**

### 主题与外观
- [ ] 内置 3 套主题：Light / Dark / Sepia
- [ ] 字体设置（CJK + 英文双字体）
- [ ] 行宽 / 字号 / 行高调节

### 图片管理
- [ ] 拖拽图片插入
- [ ] **粘贴图片自动写入 vault 子目录**（assets/ 或自定义；Typora 同款）
- [ ] 图片相对路径

## P2 — 可以（M3 / V1.0 候选）

- [ ] 导出 PDF（用 WKWebView 打印对话框 + 自定义 CSS）
- [ ] 导出 HTML（带样式）
- [ ] 导出 docx（用 pandoc 子进程，需要用户安装 pandoc，或内置 wasm 版）
- [ ] 拼写检查（macOS NSSpellChecker via Swift bridge，或纯 JS）
- [ ] 多窗口
- [ ] 自定义快捷键
- [ ] 自定义 CSS 主题加载
- [ ] 段落锚点（双击段落复制段落链接）
- [ ] PDF 内嵌预览
- [ ] [[wikilink]] 解析（保存为元数据，UI 暂不利用）

## P3 — 不做（V1.x 后再议）

- ❌ 双向链接图谱
- ❌ 标签系统
- ❌ 数据库视图 / 属性
- ❌ 实时协作
- ❌ 插件市场
- ❌ AI 助手
- ❌ Git 集成（用户可在外部用）
- ❌ 多端同步（用户用 iCloud / Dropbox）
- ❌ Window / Linux 版

## 验收 Demo 脚本（M1 结束时跑）

1. 双击 markup.app，1.5 秒内出现窗口
2. ⌘O 打开一个 5MB 的真实笔记，500ms 内渲染完成
3. 在文档中段落连续输入 3 秒，无卡顿
4. ⌘O 打开一个 1 万 .md 文件的 vault，文件树显示流畅
5. 等待索引完成（< 5s 提示），⌘⇧F 搜索一个生僻词，结果毫秒级返回
6. 用源码模式开同一文件，切回 WYSIWYG，内容无丢失
7. 退出，重启，最近文件还在；活动监视器看常驻内存 < 300MB

## 风险点（基于报告 04）

| 风险 | 缓解 |
|------|------|
| Milkdown 在 5MB 文档下卡顿 | M0 spike 验证；如失败 → 改用裸 ProseMirror + 自写 lazy rendering |
| WKWebView 中文 IME 问题 | M0 spike 验证；如失败 → 上 Cocoa input adapter（Tauri issue 已有方案） |
| Tantivy 索引大文件超时 | 跳过 > 1MB 单文件；用户可白名单 |
| 公证流程踩坑 | M0 就跑通空壳公证；不要 V1 临门一脚才发现签名问题 |
