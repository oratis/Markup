# Welcome to Markup

This is a sample [[features]] vault — a plain folder of Markdown files. Markup
opens **your** folders the same way; nothing is imported and nothing leaves your
device.

## Try this

- Tap or click **[[features]]** to follow a wikilink.
- Search for the word **callout** to see full-text search.
- Switch to **Edit** and type — the file on disk stays plain Markdown.
