## Description
```
Markup is a fast, native, open-source Markdown editor for macOS. Instead of a
plain text box, it renders your notes like a clean web page — and you edit on
demand. Built on Tauri (Rust + the system WebView), so it's genuinely native
and light (~88 MB), not a heavy Electron app.

• Reader-first — your Markdown rendered like a document, edit with one keystroke
• A real vault — wikilinks, backlinks, a graph view, and full-text search
  (Tantivy indexes a 10,000-file vault in about a second)
• Markdown → HTML in one click — a themed, self-contained page that keeps your
  code highlighting, KaTeX math, and Mermaid diagrams; print to PDF too
• Free and open source (MIT). No account, no telemetry.

Download (Apple Silicon / Intel) — signed & notarized, opens with no Gatekeeper prompt:
https://github.com/oratis/Markup/releases/latest
```
